## Core Features

### Language Server Protocol Support

Built-in LSP configuration with Mason for automatic installation and management of language servers:

- TypeScript/JavaScript (ts_ls, eslint)
- Python (pyright, pylint)
- Web technologies (HTML, CSS, Tailwind, Svelte)
- Lua (lua_ls)
- GraphQL, Prisma, and more

<Note>
All language servers are automatically installed on first launch via Mason, so you can start coding immediately.
</Note>

### Smart Auto-Completion

Intelligent completion powered by nvim-cmp with multiple sources:

- LSP-based code completion
- Buffer text suggestions
- File path completion
- Snippet expansion with LuaSnip

<Tip>
Use `Tab` to cycle through suggestions and `Ctrl+Space` to trigger completion manually.
</Tip>

### Quick File Navigation

**Telescope Fuzzy Finder:**
- `Space Space` - Find files in current directory
- `Space /` - Live grep search across your project
- `Space fr` - Browse recent files
- `Space fk` - Search keymaps

**Harpoon for Instant Jumps:**
- `Space H` - Mark current file
- `Space h` - Open harpoon quick menu
- `Space 1-9` - Jump to marked file by number

### Syntax Highlighting

Treesitter provides advanced syntax highlighting and code understanding for 29+ languages including:

### Automatic Formatting

Conform.nvim handles formatting with support for:
- Prettier (JS/TS/CSS/HTML/JSON/Markdown)
- Stylua (Lua)
- Black & isort (Python)

<Info>
Formatting runs automatically on save, keeping your code clean without manual intervention.
</Info>

### Git Integration

**Neogit** provides a Magit-like interface for Git operations:
- `Space gg` - Open Neogit in floating window

**Diffview** for visual diff inspection:
- `Space dd` - Open diff view
- `Space ds` - Compare your branch with origin/main
- `Space df` - Close diff view
