## Default Theme

The default theme is set to **Bamboo**:

```lua lua/leyland/plugins/colorscheme.lua
config = function()
  vim.cmd.colorscheme("bamboo")
end
```

## Switching Themes

To change your colorscheme, use the `:colorscheme` command:

<Tabs>
  <Tab title="Catppuccin">
    ```vim
    :colorscheme catppuccin
    :colorscheme catppuccin-mocha
    :colorscheme catppuccin-macchiato
    :colorscheme catppuccin-frappe
    :colorscheme catppuccin-latte
    ```
  </Tab>
  
  <Tab title="Tokyo Night">
    ```vim
    :colorscheme tokyonight
    :colorscheme tokyonight-night
    :colorscheme tokyonight-storm
    :colorscheme tokyonight-day
    :colorscheme tokyonight-moon
    ```
  </Tab>
  
  <Tab title="Other Themes">
    ```vim

    :colorscheme rose-pine
    :colorscheme rose-pine-moon
    :colorscheme rose-pine-dawn
    :colorscheme onedark
    :colorscheme modus
    :colorscheme modus_vivendi
    ```

  </Tab>
</Tabs>

## Making Your Choice Permanent

To set your preferred theme as the default:

<Steps>
  <Step title="Open the colorscheme plugin file">
    Navigate to `lua/leyland/plugins/colorscheme.lua`
  </Step>
  
  <Step title="Modify the config function">
    Change the colorscheme name in the Rose Pine config:
    ```lua
    config = function()
      vim.cmd.colorscheme("your-theme-here")
    end
    ```
  </Step>
  
  <Step title="Restart Neovim">
    Restart Neovim or run `:source %` to apply changes
  </Step>
</Steps>

## Configuration

All themes are loaded with `priority = 1000`, ensuring they load before other plugins that might depend on colors.

```lua
priority = 1000
```

The Rose Pine plugin includes the default colorscheme configuration because it's defined last.

## Customizing Themes

Many themes support additional configuration. For example, to customize OneDark:

```lua
{
  "navarasu/onedark.nvim",
  priority = 1000,
  config = function()
    require("onedark").setup({
      style = "deep",              -- Choose style
      transparent = false,         -- Disable transparency
      term_colors = true,          -- Terminal colors
      ending_tildes = false,       -- Hide tildes
      code_style = {
        comments = 'italic',
        keywords = 'bold',
        functions = 'none',
        strings = 'none',
        variables = 'none'
      },
    })
    require("onedark").load()
  end,
}
```

<Tip>
Refer to each theme's GitHub repository for detailed configuration options and customization guides.
</Tip>

## Theme Previews

To quickly preview themes without restarting:

1. Use `:colorscheme <theme-name>`
2. If you don't like it, try another
3. Once you find one you like, make it permanent

<Note>
Some theme changes may require a restart to fully apply, especially for UI elements like statusline and tabline.
</Note>
