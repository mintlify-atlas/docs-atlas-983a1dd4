## Quick Start

All plugins are automatically installed and configured when you first launch nvim-luffy. The plugin manager (lazy.nvim) will download and set up everything for you.

<Tip>
Press `<leader>fk` in normal mode to open Telescope's keymap finder and explore all available plugin keybindings.
</Tip>

## Plugin Manager

nvim-luffy uses [lazy.nvim](https://github.com/folke/lazy.nvim) as its plugin manager, which provides:

- Lazy loading for fast startup times
- Automatic plugin installation
- Lock file for reproducible configurations
- Easy plugin updates
