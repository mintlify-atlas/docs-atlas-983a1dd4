Complete reference of all commands available in nvim-luffy.

## Built-in Commands

These are standard Neovim commands used in the configuration:

### Buffer Management

| Command | Description | Source |
|---------|-------------|--------|
| `:bd` | Close current buffer | core/keymaps.lua:14 |
| `:%bd` | Close all buffers | core/keymaps.lua:15 |

### LSP Commands

| Command | Description | Source |
|---------|-------------|--------|
| `:LspRestart` | Restart the LSP server | lsp.lua:51 |

## Plugin Commands

### File Explorer (nvim-tree)

| Command | Description | Source |
|---------|-------------|--------|
| `:NvimTreeFindFileToggle` | Toggle file explorer and find current file | plugins/nvim-tree.lua:5 |
| `:NvimTreeOpen` | Open file explorer | nvim-tree plugin |
| `:NvimTreeClose` | Close file explorer | nvim-tree plugin |
| `:NvimTreeFocus` | Focus on file explorer | nvim-tree plugin |
| `:NvimTreeFindFile` | Find current file in tree | nvim-tree plugin |
| `:NvimTreeCollapse` | Collapse file explorer tree | nvim-tree plugin |
| `:NvimTreeRefresh` | Refresh file explorer | nvim-tree plugin |
