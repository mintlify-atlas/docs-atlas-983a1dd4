Complete keymap reference for nvim-luffy. Leader key is set to `<Space>`.

## General Keymaps

| Mode | Key | Action | Description | Source |
|------|-----|--------|-------------|---------|
| n | `<leader>\|` | `<C-w>v` | Split window vertically | core/keymaps.lua:11 |
| n | `<leader>sh` | `<C-w>s` | Split window horizontally | core/keymaps.lua:12 |
| n | `<leader>bd` | `:bd` | Close buffer | core/keymaps.lua:14 |
| n | `<leader>ba` | `:%bd` | Close all buffers | core/keymaps.lua:15 |

## Mode Abbreviations

- `n` - Normal mode
- `i` - Insert mode
- `v` - Visual mode
- `x` - Visual mode (exclusive)
- `o` - Operator-pending mode
- `c` - Command-line mode
